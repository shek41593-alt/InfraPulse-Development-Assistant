# InfraPulse

InfraPulse is a modular Linux infrastructure monitoring toolkit written in Bash.

## Features

- System Monitoring
- Service Monitoring
- Security Auditing
- Log Analysis
- Multi-Server Monitoring
- HTML & JSON Reports
- Docker Support
- Ansible Integration
- Interactive Dashboard

## Project Structure

config/        Configuration files
dashboard/     Terminal UI
lib/           Shared Bash libraries
monitoring/    Monitoring modules
reports/       Generated reports
logs/          Application logs
